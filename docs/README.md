# Portfolio Documentation

This directory contains the maintained directives for Portfolio. A directive
is the single repository authority for its domain and describes the behavior
that is implemented now.

- [development.md](./development.md) — workspace development, dependency
  modes, environment ownership, and Agentime client integration.
- [testing.md](./testing.md) — test layers, placement, tooling, and canonical
  verification commands.
- [deployment.md](./deployment.md) — production promotion, dependency
  installation, build, restart, and health qualification.

Repository-wide worktree rules and directive routing live in
[AGENTS.md](../AGENTS.md). Product details belong in source-level documentation
or the owning directive rather than being duplicated in this index.
